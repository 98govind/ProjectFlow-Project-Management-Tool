# ProjectFlow - Project Management Tool(live link : https://projectflow-d571.onrender.com/login )

ProjectFlow is a full-stack project management web application built using Flask, SQLite, HTML, CSS, JavaScript, and Chart.js.

It helps users create projects, manage tasks, track deadlines, monitor progress, and analyze project performance through interactive dashboards.

---

## Features

- User Authentication (Register/Login/Logout)
- Create, Edit, Delete Projects
- Create, Edit, Delete Tasks
- Task Status Management (To Do, In Progress, Done)
- Task Priority Management
- Deadline Tracking
- Overdue & Due Soon Detection
- Project Progress Tracking
- Search & Filter Projects
- Search & Filter Tasks
- Dashboard Analytics
- Pie Chart & Bar Chart Visualization
- Task Comments System
- Activity Logs

## Installation

### Clone Repository

```bash
git clone https://github.com/98govind/ProjectFlow-Project-Management-Tool.git
```

### Go to Project Folder

```bash
cd ProjectFlow
```

### Install Dependencies

```bash
pip install flask flask_sqlalchemy flask_login werkzeug
```

### Run Project

```bash
python app.py
```
